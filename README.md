## Project Overview

This AI Chat App allows users to interact with an intelligent assistant through a responsive and user-friendly chat interface.  
The backend handles chat logic, data persistence and communication with the OpenAI API, while the frontend delivers a sleek UI with real-time interactions.

Through this project, I gained hands-on experience in building a **production-ready fullstack application**, managing API security, handling async data flows, and structuring scalable codebases.

## Features

- AI-powered chat using OpenAI GPT-4
- Real-time messaging experience
- Persistent chat history
- Secure API key handling
- Django REST API backend
- Modern React frontend
- Loading and error state handling
- Scalable and deployment-ready architecture
